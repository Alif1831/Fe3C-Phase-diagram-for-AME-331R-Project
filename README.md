# Fe3C Phase Diagram for AME 331R Project

Interactive steel heat-treatment and phase-transformation simulator for visualizing:

- TTT and CCT style diagrams
- cooling paths for annealing, normalizing, and quench + temper
- phase evolution along the path
- microstructure and property changes

## Live site

After GitHub Pages is enabled, the site can be viewed at:

`https://alif1831.github.io/Fe3C-Phase-diagram-for-AME-331R-Project/`

## Files

- `index.html` — main interactive simulator

## Features

- Adjustable carbon percentage
- Heat-treatment selection:
  - Annealing
  - Normalizing
  - Quench + Temper
  - Spheroidizing
- TTT / CCT diagram toggle
- Cooling-rate control
- Tempering temperature control
- End-temperature control
- Interactive phase and microstructure display

## How to run locally

1. Download or clone the repository
2. Open `index.html` in your browser

## How to publish with GitHub Pages

1. Push the project to your GitHub repository
2. Go to **Settings**
3. Open **Pages**
4. Under **Build and deployment**, choose **Deploy from a branch**
5. Select `main`
6. Select `/(root)`
7. Click **Save**

GitHub will then publish the site from your `index.html`.

## Project purpose

This simulator was created for an AME 331R project to help visualize steel phase transformations and heat-treatment behavior in an interactive format.

## Notes

This is an educational visualization tool. The curves and phase behavior are simplified for teaching and demonstration.
